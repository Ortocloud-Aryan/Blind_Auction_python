# 🏆 Blind Auction Program (Python)

A simple command-line **Blind Auction system** built using Python.
The program collects bidders' names and their bidding amounts, stores them in a dictionary, and determines the winner with the highest bid.

## 🚀 Features

* Accepts multiple bidders and their bids
* Stores bidder data using Python dictionaries
* Uses loops and conditional logic to manage the bidding process
* Automatically determines the highest bidder using Python's `max()` function

## 🧠 Concepts Used

* Dictionaries
* Functions
* Loops (`while`)
* Conditional statements (`if`)
* Built-in functions (`max()` with `key` parameter)
* User input handling

## ▶️ How It Works

1. The program asks for the bidder's **name**.
2. It then asks for the **bid amount**.
3. The user decides whether more bidders want to participate.
4. When bidding stops, the program calculates and prints the **highest bidder and their bid**.
